
## xba

**Exchange B and A Accumulators**

Swaps the high and low bytes of the Accumulator

#### Affected Flags

```
N-----Z--
```

#### Modes

| OpCode | Syntax | Addressing | Bytes | Cycles     |
|--------|--------|------------|-------|------------|
| `0xEB` | `xba`  | Implied    | 1     | 3          |
